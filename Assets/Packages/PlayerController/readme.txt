Asset основан на пакете kinematic character controller.

Устройство персонажа на примере объекта Player:

Player

-Character
--Root        - Содержит весь визуал персонажа.
--CameraPoint - Точка за которой следует камера (к которой крепится), может менять положение.

-CameraController - вращается сам при помощи мыши для осмотра.
--CameraHolder    - нужен для изменения оси вращения камеры (можно менять локальную позицию), например, для иммитации головы.
---CameraAnimator - нужен для локальной анимации, например, покачивания камеры.
----Camera        - содержит сам компонент camera, можно свободно трансформировать, например, применять эффекты тряски.

Player не меняет своего положения при перемещении, но он может задать начальное положение. 
Перемещается по миру только Character, а CameraController следует за CameraPoint.
Для удобства работы с transform персонажа и управления взглядом, 
реализован класс KinematicEntity, от которого наследуется Player.

Подобным образом можно реализовать и управление персонажем с помощью ИИ, для этого приведен пример ExampleAIController.
